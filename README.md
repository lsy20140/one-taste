## 서비스 소개

***OneTaste** = **One(하나뿐인)** + **Taste(맛)***

하나뿐인 맛을 경험하고 싶은 사람들을 위해 제작한 비 프랜차이즈 맛집 지도입니다.




<br>
<br>
